* i3

- Mod1 Alt key

- Mod4 win key


** move

| key     | direction   |
|---------+-------------|
| $mod+j  | left        |
| $mod+k  | down        |
| $mod+l  | up          |
| $mod+;  | right       |


** layout

$mod+e splith/splitv 切换布局

$mod+s stacking 变成一层层堆积的栈状，有焦点的显示在最上方

$mod+w tabbed 与上面类似，但变成了tabbed状

** fullscreen

$mod+f

** quit

$mod+Shift+q

** workspace

$mod+num 数字键为workspace
